# 🌍 Moon Orbit Animation

A minimalist, pure HTML and CSS animation depicting the Moon orbiting around the Earth. This project uses absolute positioning, flexbox centering, CSS variables, and `@keyframes` rotations to achieve a smooth, infinite orbital motion.

---

## 🚀 Features

* **Perfect Flexbox Centering:** The space canvas is dynamically centered within the viewport.
* **Absolute Positioning & Translation:** Uses the `transform: translate(-50%, -50%)` trick to anchor elements flawlessly to their parent's center.
* **Infinite CSS Animation:** Smooth, linear $360^\circ$ rotation loops every 5 seconds.
* **Semantic Layout:** Clean, minimal DOM structure (`.space` -> `.earth` & `.orbit` -> `.moon`).
